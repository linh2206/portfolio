# LE HOAI LINH - Portfolio

> **View Live Portfolio:** [https://linh2206.github.io/portfolio/](https://linh2206.github.io/portfolio/)

Personal portfolio website showcasing experience, projects, and skills as a Full-stack Developer.

## Tech Stack

- **Framework:** Next.js 16 (App Router, Turbopack)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Fonts:** DM Sans, JetBrains Mono

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
